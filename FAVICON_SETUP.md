# Favicon Setup

The site needs a favicon. You can:

## Option 1: Quick Fix (Create Simple Favicon)
1. Go to https://favicon.io/favicon-generator/
2. Create a simple favicon with "DA" text (David Araj initials)
3. Download the favicon package
4. Place `favicon.ico` in the `public` folder

## Option 2: Use Existing Image
If you have a team logo or player photo:
1. Use an online converter like https://favicon.io/favicon-converter/
2. Upload your image
3. Download and place `favicon.ico` in `public` folder

## Option 3: Next.js App Icon (Recommended)
Create a file called `app/icon.png` or `app/favicon.ico`:
- Size: 32x32 or 16x16 pixels
- Next.js will automatically serve it
